## Core Features

### Dashboard Analytics
Comprehensive overview of business performance powered by dynamic charts (Recharts).

### Customer Management
Full CRUD support, filtering, pagination, and insightful stats (email domain, new weekly customers).

### Employee Management
Position filtering, status labels, gender & role distribution charts.

### Menu Management
Image upload, category filters, sorting, availability toggles.

### Transaction Management
Multi-item selection, automatic total calculation, payment method insights, printable receipts.

### Modern UI & Experience
- Smooth page transitions  
- Glassmorphism effects  
- Scroll-based animations  
- Consistent design system
